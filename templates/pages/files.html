Cron Scripts (.sh)

  • • Bash/shell scripts for cron jobs
  • • Must start with shebang (#!/bin/bash)
  • • Executable permissions automatically set
  • • Can be selected when creating cron jobs

Systemd Units (.service)

  • • Service configuration files
  • • Must contain [Unit] or [Service] section
  • • Automatically synced to /etc/systemd/system
  • • Can be selected when creating services
